# HTML Elements Explanation:
- <h1> to <h6>: Heading tags used for organizing content hierarchy.
- <ul> and <ol>: Used for listing skills and education.
- <table>: Used to display work/projects in a structured format.
- <form>: Collects user input for contact.
- <input>, <textarea>, <select>: Collects various input types.
- <main>, <section>: Used for semantic layout.

# Why I chose specific input types:
- type="email": Ensures valid email input.
- type="tel": Allows numeric phone input with pattern validation.
- type="date": Easy to input DOB.
- type="range": Simple way to rate experience.

# Navigation:
- All pages have a navigation bar linking to each other.
- About page contains both internal (portfolio) and external (GitHub) links.
